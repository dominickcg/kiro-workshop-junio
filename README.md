# Kiro Workshop — Guía Completa de Funcionalidades

> Documentación de referencia sobre todas las funcionalidades de [Kiro](https://kiro.dev), el IDE agéntico de AWS que lleva rigor de ingeniería al desarrollo con IA.

## ¿Qué es Kiro?

Kiro es un entorno de desarrollo impulsado por IA que ayuda a los equipos de ingeniería a cerrar la brecha entre el "vibe coding" y el código de producción. Combina chat multimodal, desarrollo guiado por especificaciones, automatización con hooks y extensibilidad mediante MCP y Powers en una experiencia de desarrollo familiar pero potenciada por agentes.
